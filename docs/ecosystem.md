# LuminaryIoTChain 与 LuminaryWorks AI 生态

> 对外说明 · 技术规格见 [spec/ecosystem.md](../spec/ecosystem.md)

## 本产品是什么

**LuminaryIoTChain** 是开源、AI 驱动的 IoT PaaS：EMQX + ThingsBoard + 控制台，帮助硬件厂商**低于涂鸦成本**完成设备上云。可**单独私有化部署**。

## 在 LuminaryWorks 五项目中的位置

| 产品 | 角色 |
|------|------|
| DataLuminary | **看见** — DataTalk 设备监控大屏 |
| VibeEdu | **学** — 工程师 ESPHome/MQTT 接入辅导 |
| **LuminaryIoTChain（本产品）** | **连** — 设备 MQTT 管道与多租户平台 |
| VistaRemote | **控** — 设备远程运维 |
| VibeAgent | **赚** — 设备 Agent 链上变现 |

```text
设备 ──MQTT──► LuminaryIoTChain ──► 可选跳转四兄弟产品
```

## 与涂鸦的差异（摘要）

- 开源可私有化，许可与部署成本更低  
- 不止设备管理：**AI 推理、DataTalk 大屏、链上 Agent 市场**  
- VibeEdu 降低工程师接入门槛
